# @mastors/flexer

> Complete flexbox utility class system for the Mastors ecosystem.

## Installation

```bash
npm install @mastors/flexer
```

Requires `@mastors/core` as a peer dependency.

## SCSS Usage

```scss
@use "@mastors/flexer/scss";
```
