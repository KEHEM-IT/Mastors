# mastors

> Mastors – A professional-grade, scalable frontend SCSS utility ecosystem.

[![npm](https://img.shields.io/npm/v/mastors.svg)](https://www.npmjs.com/package/mastors)
[![license](https://img.shields.io/badge/license-MIT-blue.svg)](../../LICENSE)

## Install

```bash
npm install mastors
```

Installs `@mastors/core` by default. After install, run the interactive picker to add more packages:

```bash
npx mastors
```

## Packages

| Package | Description |
|---|---|
| `@mastors/core` | Tokens, mixins, functions & SCSS architecture *(included)* |
| `@mastors/gridder` | CSS Grid utility classes |
| `@mastors/flexer` | Flexbox utility classes |

## Install packages individually

```bash
npm install @mastors/core
npm install @mastors/gridder
npm install @mastors/flexer
```
